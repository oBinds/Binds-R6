Features

Menu Navigation
ALL PAGES
Open Menu
L2+OPTIONS or LT+MENU

Hold L2 and press Options to open the menu. UP/DOWN scrolls the category list (Op & Gun, Anti-Recoil, Rapid Fire, Movement, Misc). CROSS (A) enters a category. LEFT/RIGHT D-Pad scrolls through pages inside the selected category. CIRCLE (B) goes back. All settings save automatically when you close the menu. The menu also has an R6 Miscs tab at the top level for global options.

Op & Gun
PAGE 1 — ATK & DEF
Operator Select
UP / DOWN

Scroll UP/DOWN on the Operator page to change which operator's settings are active. The script remembers your last selected operator for both ATK and DEF separately — it loads back automatically on power on. 39 Attackers and 38 Defenders supported.

PAGE 2 — ATK & DEF
Gun Selection
UP/DOWNbrowse✕select or A

Scroll UP/DOWN to browse every gun assigned to the current operator — browsing alone never changes any values. Press ✕ (Cross / A) on the gun you have equipped in-game to select it. A * appears next to the currently selected gun. Primary and secondary are selected independently. Every gun has its own saved vert/horiz/RF — selecting loads the correct values instantly.

Anti-Recoil
PAGES 3–4 — ATK & DEF
AR Vertical & Horizontal
UP / DOWN

Every gun has its own Vertical (up/down recoil pull) and Horizontal (left/right drift) values saved permanently to the Zen. Adjust UP/DOWN to tune. Values are saved per gun globally — so MPX is always the same value on every operator that uses it. All values come pre-tuned from the script defaults and only need adjusting if you want something different.

Rapid Fire
PAGE 5 — ATK & DEF
Rapid Fire
UP=ONDOWN=OFF

Automatically spams R2 / RT at the configured speed while you hold the trigger — turns any semi-auto into a full-auto. Pre-set ON for semi guns: OTS-03, CAMRS, AR-15.50, D-50, 417, SDP 9mm, PMR90A2, TACIT .45, P226 MK 25, P9, CAMRS, LFP586, Mk14 EBR and more. Pre-set OFF for all full-auto guns. Can be toggled per slot.

PAGE 6
RF Speed
UP / DOWN

Controls how fast rapid fire spams. Higher = faster presses. Default is pre-tuned for most semi guns. Shown as a bar graph on the OLED. Adjust to match your gun's fire rate for best results.

PAGE 7
RF For All
TOGGLE

Forces Rapid Fire ON for every gun regardless of individual settings. Useful when you want RF on a full-auto gun temporarily. Toggle ON/OFF — overrides per-slot RF settings while active.

PAGE 8
Skel Key RF
TOGGLE

Skeleton Key RF — spams R1 / RB instead of R2. For operators using the under-barrel shotgun (Skeleton Key attachment). Fires the under-barrel weapon rapidly using R1 instead of the main trigger. Enable separately from main RF.

Movement
PAGES 9–10
Crouch Spam
TOGGLE + SPEED

Rapidly crouches and stands while you're shooting — makes you a much harder target to hit. Active while holding R2 / RT. Speed is adjustable on the next page (Crouch Spd). Great for gunfights in tight angles.

PAGES 11–12
Strafe
TOGGLE + SPEED

Automatically moves left and right repeatedly while firing — creates unpredictable lateral movement making you harder to track. Works while holding R2 / RT. Strafe Spd controls the speed of the side-to-side motion.

PAGES 13–14
Shaiko Lean
L2+R2hold or LT+RThold

While holding L2 (ADS) and R2 (shoot), continuously alternates between leaning left (L3) and leaning right (R3) — creates rapid side-to-side lean peeking. Speed adjusted with Shaiko Spd. Also controls speed for Stick Lean. Toggle in menu to enable.

PAGES 15–16
Lean Spam
L2+R2hold

Rapidly spams the R1/L1 lean buttons while holding L2+R2. Different to Shaiko — uses the shoulder lean buttons rather than stick clicks, giving a different feel. Lean Speed slider controls the rate. Good for defenders holding angles.

PAGE 17
Prone Shot
TOGGLE

Automatically goes prone the moment you press R2 / RT to shoot, then stands back up when you release. Catches enemies off guard as you drop to the floor while firing. Pairs well with anti-recoil for prone spray.

PAGES 18–19
Tea Bag
TOGGLE + SPEED

Rapidly spam crouches on command — a pure crouch spam not tied to shooting. Great for tea-bagging or mixing into gunfights as a distraction. TeaBag Spd controls how fast it crouches/stands.

PAGE 20
Auto Lean
TOGGLE

Automatically leans while ADS. When you hold L2 the script adds a lean input so you're always slightly peeking. Can be used to hold pre-aimed corners — lean out to shoot without manually pressing lean buttons.

PAGE 21
Hip Lean
TOGGLE

Automatically leans while hip-firing (not ADS). Adds a lean input when you press R2 without L2. Useful for hip-fire spraying around corners — you lean slightly while you shoot without touching the lean button.

Misc
PAGE 22
Dead Zone
UP / DOWN

Adjusts the controller analog stick deadzone. Lower = more responsive sticks that react to smaller inputs. Higher = ignores micro-movements. Helps fine-tune aim sensitivity for your personal playstyle.

PAGE 23
On Cams
TOGGLE

Keeps anti-recoil active while using operator cameras (e.g. Valkyrie cams, Maestro Evil Eye). Normally AR turns off when not directly shooting — this keeps it running while you're watching cameras and shooting from them.

PAGE 24
Pin On Shot
TOGGLE

Locks your aim position the moment you start firing — prevents stick drift from moving your crosshair during a spray. Pins the aim in place when R2 is pressed so your recoil correction lands exactly where intended.

PAGE 25
Auto Scan
TOGGLE

Automatically activates scanning when the feature is on — useful for operators with scan abilities like Jackal (tracking) or Lion (scan pulse). Fires the ability input automatically so you don't have to manually trigger it.

PAGE 26
Inverted AR
TOGGLE

Flips the direction of the anti-recoil compensation. If vertical recoil normally pulls the stick down to counteract upward kick, this inverts it — pulls up instead. For operators or guns where standard AR feels off.

PAGE 27
Abilities
TOGGLE

Activates operator ability automation. When enabled, automatically uses or spams the operator's gadget. Useful for operators where holding the ability button is needed — lets you focus purely on aim and movement.

PAGE 28
LED
TOGGLE

Toggles the Cronus Zen LED indicator on or off. When OFF the light stays dark regardless of script activity. Useful if you want to keep the device less visible.

PAGE 29 — ATK / PAGE 57 — DEF
Stick Lean
L2+L3/L2+R3 or LT+LS/LT+RS

Tap L2+L3 or L2+R3 — works as a completely normal single lean. Hold L2+L3 past 250ms — spams lean left continuously. Hold L2+R3 past 250ms — spams lean right. Speed uses Shaiko Spd. ATK and DEF independent.

R6 Miscs
PAGE 58 — R6 MISCS
Soft Triggers
MENU ONLY

Any touch on R2 / RT or L2 / LT — even the lightest press — instantly fires at 100%. Eliminates the trigger deadzone completely for the fastest possible ADS and fire response. ON by default. Access via the R6 Miscs button on the category screen (not inside Attackers/Defenders).

Tips & Usage

  • First time setup: Clear the Zen's persistent memory before loading the script for the first time. This lets all the pre-tuned gun defaults load cleanly from scratch.
  • Selecting your gun: Every time you change your in-game loadout, open the menu, go to your op's Gun page and press ✕ on the gun you equipped. The script loads the correct vert/horiz/RF instantly and saves it.
  • Same gun = same value everywhere: MPX is the same tuned value on Valkyrie, Warden, and Tubarao. Tweaking it once on any of them updates it for all three.
  • Shaiko vs Stick Lean: Shaiko (Movement) alternates L3 / LS + R3 / RS while you hold L2+R2 / LT+RT — both directions. Stick Lean (Misc) spams one direction only — L2+L3 / LT+LS = only left, L2+R3 / LT+RS = only right. Use Stick Lean for aggressive one-sided peeking.
  • Movement features stack: Crouch Spam and Strafe can run together. Shaiko and Crouch Spam can run at the same time too. Experiment with combining them for unpredictable gunfight movement.
  • Rapid Fire is pre-tuned: Some if not all semi guns come with RF ON at the right speed. Full-auto guns come with RF OFF. You never have to manually set these — just select your gun and it's done.
  • Soft Triggers are ON by default. The lightest touch of L2 or R2 fires at 100%. If you want more analog control over ADS speed, turn it off in R6 Miscs.
  • ATK and DEF are fully independent. Every single setting — recoil, RF, movement features, speed sliders, stick lean, soft triggers — can be different per side. ATK changes never affect DEF and vice versa.
  • Everything auto-saves. Close the menu and all your changes write to the Zen's persistent memory. Next power on loads everything back exactly as you left it.
